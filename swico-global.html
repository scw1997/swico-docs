Global配置 ​

Global配置是在模板的src/global.ts文件中用于定义一些全局配置性的参数或方法（需要默认导出）

推荐使用defineGlobal api来获得更好的TypeScript类型提示。

支持的可配置项如下：

Vue模板 ​

onInit ​

Vue模板挂载的App实例初始完成后的回调。

返回两个参数：app（vue应用实例）和router（vue-router对象）。

可以在此对app和router进行api调用操作，比如添加插件，设置路由守卫等。

示例：

ts
// src/global.ts

import { defineGlobal } from 'swico';
import { createPinia } from 'pinia';

export default defineGlobal({
    onInit: (app, router) => {
        //注入全局状态管理store
        app.use(createPinia());
        //设置路由守卫
        router.beforeEach((to, from, next) => {
            //
        })
    }
});

React模板 ​

暂无可支持的配置项。
